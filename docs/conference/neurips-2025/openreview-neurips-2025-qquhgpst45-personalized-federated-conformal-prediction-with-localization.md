## Abstract
Personalized federated learning addresses data heterogeneity across distributed agents but lacks uncertainty quantification that is both agent-specific and instance-specific, which is a critical requirement for risk-sensitive applications. We propose personalized federated conformal prediction (PFCP), a novel framework that combines personalized federated learning with conformal prediction to provide statistically valid agent-personalized prediction sets with instance-localization. By leveraging privacy-preserving knowledge transfer from other source agents, PFCP ensures marginal coverage guarantees for target agents while significantly improving conditional coverage performance on individual test instances, which has been validated by extensive experiments.

### 2. 论文提出的方法论

论文的核心方法论分为两个层次：第一层是基于纯本地数据的广义局部共形预测（GLCP）；第二层是结合联邦学习的个性化联邦共形预测（PFCP）。

*   **2.1 广义局部共形预测 (GLCP)**
    *   **核心思想**：仅使用目标代理自己的数据。它通过估计得分S在给定输入x下的条件分布F(s|x)，将原始共形得分S转化为一个位于\[0,1] 区间的“变换得分”V（本质上是条件概率积分变换）。这个变换使得预测集能自然地实现实例级的局部化适应。
    *   **关键技术细节**：
        1.  使用一个通用条件分布估计算法A（如论文中提到的Engression），在本地训练集上学习一个估计量 `F_hat(s|x)`。
        2.  对于校准集和测试点，计算变换得分 `V_i = F_hat(S_i | X_i)`。
        3.  基于变换得分V，使用标准的共形预测框架构建预测集：包含所有使得变换得分不超过校准集得分经验分位数的标签y。
    *   **意义**：GLCP作为一个统一框架，通过选用不同的`F_hat`估计器，可以泛化现有的多种局部共形预测方法（如Localized CP, Distributional CP）。

*   **2.2 个性化联邦共形预测 (PFCP)**
    *   **核心思想**：GLCP的性能高度依赖于本地对`F(s|x)`的估计精度。当目标代理数据有限时，估计不准。PFCP通过联邦学习，利用其他源代理的数据来**增强**对目标代理条件分布`F(s|x)`的估计，从而在保证边际覆盖率不受影响的前提下，显著提升预测集的实例级性能。
    *   **方法论流程（三步走策略）**：
        1.  **联邦估计条件分布**：除了在目标代理本地估计`F_hat(s|x)`外，使用联邦学习算法（如FedAvg），在K个源代理上协作训练一个**混合条件分布** `F_hat_mix(s|x)`。
        2.  **联邦估计密度比**：将估计源代理混合分布与目标代理分布之间的**联合密度比** `r(x, s) = f(x,s) / f_mix(x,s)`建模为一个二分类问题。使用联邦平均算法训练一个分类器来区分样本来自目标代理还是混合源代理，并通过Platt校准得到最终的密度比估计量 `r_hat(x,s)`。
        3.  **聚合增强估计量**：利用一个巧妙的等价变换 `F(s|x) = E_{S~F_mix(s|x)}[1(S <= s) * r(S|x)]`，将本地估计`F_hat(s|x)`和联邦估计`F_hat_mix(s|x)`与`r_hat(x,s)`进行加权聚合，得到增强的条件分布估计量 `F_agg(s|x)`。聚合公式为：
            `F_agg(s|x) = {1 + r_hat(x)}^{-1} * [E_{S~F_hat(s|x)}1(S ≤ s) + E_{S~F_hat_mix(s|x)}{1(S ≤ s) * r_hat(x, S)}]`。
        4.  **生成预测集**：使用`F_agg(s|x)`替换GLCP中的`F_hat(s|x)`，计算新的变换得分`V_agg`，并据此构建最终的个性化联邦共形预测集。

    *   **隐私保护**：在整个过程中，只有模型参数或梯度在代理间通信，原始数据从不离开本地。
