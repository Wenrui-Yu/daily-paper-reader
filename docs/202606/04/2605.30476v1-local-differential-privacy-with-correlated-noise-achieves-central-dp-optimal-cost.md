## 摘要
我们研究在诚实但好奇的服务器存在下，对n个用户持有的值进行私密求和估计的问题。这要求在数据发布和整个服务器端计算过程中都保护隐私。因此，我们采用本地（纯）差分隐私模型，其中每个用户传输一个加噪扰动的值。众所周知，与中心化模型（仅在聚合后才添加噪声）相比，独立的本地噪声通常会导致显著的效用损失。我们证明这一差距并非根本性的。通过精心设计本地添加的噪声变量之间的关联性，我们构造了ε-差分隐私机制，其估计成本与中心化设置下可实现的最优成本相匹配，误差可任意小。

## Abstract
We study privately estimating the sum of $n$ user-held values in the presence of an honest-but-curious server. This motivates requiring privacy not only at data release but also throughout server-side computation. We therefore adopt the local (pure) differential privacy model, in which each user transmits a noise-perturbed value. It is well known that independent local noise typically incurs a substantial utility loss compared to the centralized model, where noise is added only after aggregation.   We show that this gap is not fundamental. By carefully designing correlations among the locally added noise variables, we construct $\varepsilon$-DP mechanisms whose estimation cost matches the optimal cost achievable in the centralized setting, up to an arbitrarily small error.

## 1. 研究背景与核心问题
- **背景动机**：在诚实但好奇的服务器模型下，用户希望进行私密的求和（如联邦平均），但服务器不可信。传统的中心化差分隐私（central-DP）要求服务器可信，而本地差分隐私（local-DP）虽能在用户端即加噪保护隐私，但通常带来严重的效用损失（如平方损失下误差从 O(1) 升至 O(n)）。
- **核心问题**：能否通过允许本地噪声变量之间存在相关性，来消除本地与中心差分隐私之间的效用差距？论文的核心发现是：**该差距并非固有，通过精心设计的关联噪声，本地 ε-DP 机制可以实现与中心化最优几乎相同的估计成本**。

## 2. 方法论
- **隐私定义**：采用纯 ε-差分隐私（ε-DP），要求对于任何邻近数据集（单用户数据变化不超过 Δ），相加噪声向量的联合分布满足
  \[
  P(A) \le e^\varepsilon P(A + t), \quad \forall t \in T_\Delta^n
  \]
  其中 \(T_\Delta^n\) 表示最多在一维上变动 [-Δ, Δ] 的向量集合。
- **关键思路**：
  - 将 n 维噪声向量 Z 的联合分布设计为相关，使得在聚合后总噪声的分布与中心化最优噪声分布一致。
  - 采用正交变换 \(U = S Z\)，其中 S 的第一行为 \(\frac{1}{\sqrt{n}}(1,\dots,1)\)，将总噪声方向 U₁ 与正交补空间分离。
  - **下界证明**：通过坐标变换将整体损耗表达为关于 U₁ 的一维积分，且 U₁ 的边际分布必须满足 Δ/√n 敏感度的 ε-DP，因此其成本不低于一维中心化最优成本 \(C^*_{\varepsilon,\Delta}(L)\)。
  - **可达性构造**：在旋转坐标系中取**乘积形式**的分布：
    \[
    \tilde P^S(du_1,\dots,du_n) = f_{\gamma^*}^{(\varepsilon_0,\Delta/\sqrt{n})}(u_1)\, g(u_2)\cdots g(u_n),
    \]
    其中 \(f_{\gamma^*}\) 是接近最优的一维楼梯（staircase）密度，但使用稍小隐私预算 ε₀ < ε；g 为 Laplace 等分布，分担剩余隐私预算 (ε-ε₀)/(n-1)。当 ε₀ → ε 时，总加噪和的损失逼近中心最优值。
- **楼梯密度族**：借助 Geng & Viswanath 的 staircase 机制，其最优密度形式为分段常数，可用参数 γ ∈ [0,1] 表示。

## 6. 主要结论与发现
- 本地 DP 与中心 DP 的效用差距**非固有**，而是由**独立噪声假设**造成。

- 理论结果不限于纯 ε-DP、二维或特定损失函数；对于 Rényi DP 和向量数据亦有扩展潜力。
