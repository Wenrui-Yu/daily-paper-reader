## 摘要
通信开销是可扩展分布式学习中的关键瓶颈。尽管现有方法（如局部 SGD、小批量 SGD 及其加速变体）旨在高效利用数据点，但它们仍表现出通信轮次复杂度随总样本数 $N$ 扩展的特性。在本文中，我们提出了 Local MixVR，这是一个集成局部更新与方差缩减技术以减轻局部噪声的分布式框架。我们证明，Local MixVR 是首个消除通信复杂度对 $N$ 依赖的分布式方法，实现仅随工作节点数 $M$ 扩展的复杂度。在常见的 $M<O\left(N^{1/4}\right)$ 条件下，Local MixVR 优于最先进的小批量加速 SGD 基线，弥合了分布式优化中长期存在的差距，并为通信高效训练建立了新的范式。

## Abstract
Communication overhead is a crucial bottleneck in scalable distributed learning. While existing methods aim to efficiently utilize data points, such as Local SGD, Minibatch SGD, and their accelerated variants, they still exhibit communication-round complexity that scales with the total number of samples $N$. In this paper, we introduce Local MixVR, a distributed framework that integrates local updates with variance-reduction techniques to mitigate local noise. We show that Local MixVR is the first distributed method to eliminate the dependence of communication complexity on $N$, achieving a complexity that scales only with the number of workers $M$. In common regimes where $M<O\left(N^{1/4}\right)$, Local MixVR outperforms the state-of-the-art Minibatch Accelerated SGD baseline, bridging a long-standing gap in distributed optimization and establishing a new paradigm for communication-efficient training.

### 2. 论文提出的方法论

*   **核心思想**
    *   Local MixVR 的核心是解决局部 SGD 中的“工作节点漂移”问题：即各工作节点在多次本地更新后，由于其处理不同的随机数据样本，本地模型会彼此偏离，限制了减少通信轮次的能力。
    *   该方法通过集成三种互补的方差缩减和偏差控制技术，来稳定局部轨迹、纠正同步偏差，从而允许在极其稀疏的通信条件下维持快速收敛。

*   **关键技术细节与算法流程**
    Local MixVR 在每个通信轮次内，融合了三种机制：**I. 局部双动量**、**II. 混合本地与小批量**、**III. 漂移校正**。
    *   **I. 局部双动量（Local Double-Momentum）**
        *   基于 `μ²-SGD` 算法（Dahan and Levy, 2025），每位工作节点在本地更新时应用两个动量机制：
            *   **任意时间平均（Anytime Averaging）**：对模型参数进行动量平均，使本地模型缓慢偏离上次同步的全局模型，从而稳定本地轨迹。
            *   **STORM 估计器**：一种先进的动量方差缩减技术，通过修正标准动量的陈旧梯度偏差，为每个工作节点提供更干净、一致性更强的梯度下降方向。
    *   **II. 混合本地与小批量（Budget Mixing）**
        *   引入一个混合参数 \(\alpha \in (0, 1)\)，将算力预算 \(K\) 分配给两种互补的任务：
            *   使用 \(K_{loc} = \lfloor(1-\alpha)K\rfloor\) 个样本进行本地优化步骤，以推进优化进程。
            *   预留 \(K_{avg} = \lceil \alpha K \rceil\) 个样本进行小批量平均，在同步前降低随机噪声，抑制漂移的累积。
    *   **III. 漂移校正机制（Drift Correction Mechanism）**
        *   在同步步骤 \(t\)，工作节点利用来自“小批量平均”阶段的双点梯度评估，计算一个校正项：\(\nabla f(\bar{x}_t; B_t) - \nabla f(\bar{x}_{t-1}^{(i)}; B_t)\)。
        *   该校正项衡量了本地模型梯度与全局模型梯度之间的不匹配，通过从同步后的梯度估计器中减去此不匹配，从而修正由于本地更新引入的偏差，使得工作节点以更少的累积噪声重新开始。
    *   **算法流程（简化）**
        给定初始全局模型，在每一轮通信中：
        1.  各工作节点并行的循环执行 \(K_{loc}\) 次 **`μ²-SGD`** 本地更新。
        2.  **同步**所有工作节点的模型参数。
        3.  各工作节点并行执行**累积与漂移校正**程序：
            *   在同一个小批量 \(B_t\) 上，计算全局和先前本地模型点的梯度。
            *   利用公式 \(\tilde{d}_t = (1-\beta)d_{t-1} + \beta \nabla f(\bar{x}_t; B_t) + (1-\beta)(\nabla f(\bar{x}_t; B_t) - \nabla f(\bar{x}_{t-1}^{(i)}; B_t))\) 更新梯度估计器。
        4.  **同步**所有工作节点的梯度估计器。
        5.  各工作节点使用同步后的估计器执行一次全局模型更新。

*   **公式说明（定性）**
    *   `μ²-SGD` 更新规则体现了对标准动量的偏置校正，使得梯度估计 \(d_t\) 是对真实梯度 \(\nabla f(\bar{x}_t)\) 的无偏估计，且其方差降低至 \(O(\tilde{\sigma}^2/t)\)。
    *   “漂移校正”中的校正项利用函数的平滑性（L-smoothness），将其方差约束在与工作节点漂移量 \(\|\bar{x}_t - \bar{x}_{t-1}^{(i)}\|^2\) 成比例的范围内。通过对漂移量进行仔细的解耦和界定，此部分误差得到了有效控制。
