## 摘要
协作学习只有在每个参与者都受益时才能持续。标准联邦学习优化全局平均目标，这可能导致其数据分布与总体分布显著不同的客户端表现不佳。我们研究自私个性化：一个指定的目标客户端如何利用同伴梯度来最小化自身风险，同时避免负迁移。我们提出SP-CACW，一个收敛感知的客户端加权框架，通过最小化目标客户端收敛误差的上界来选择聚合权重。所得规则明确地在同伴偏差与随机方差之间进行权衡，并且可以为有害的同伴分配零权重。我们在光滑性和有界方差假设下提供了收敛保证，并在MNIST、CIFAR-100和LEAF Shakespeare上评估该方法，该方法与强大的个性化基线和聚类基线相比具有竞争力或有所改进。

## Abstract
Collaborative learning is sustainable only when it benefits each participant. Standard federated learning optimizes a global average objective, which can under perform for clients whose data distributions differ substantially from the population. We study selfish personalization: how a designated target client can use peer gradients to minimize its own risk while avoiding negative transfer. We propose SP-CACW, a convergence-aware client-weighting framework that selects aggregation weights by minimizing an upper bound on the target client's convergence error. The resulting rule explicitly trades off peer bias against stochastic variance and can assign zero weight to harmful peers. We provide convergence guarantees under smoothness and bounded-variance assumptions and evaluate the method on MNIST, CIFAR-100, and LEAF Shakespeare, where it is competitive with or improves over strong personalized and clustering baselines.

### **2. 论文提出的方法论**

*   **核心思想**:
    *   提出 **SP-CACW** (Convergence-Aware Client Weighting for Selfish Personalized Learning) 框架。其核心理念是**直接以优化目标客户端的收敛性能为导向**来选择同伴权重，而非依赖启发式的相似度度量。
    *   通过最小化目标客户端收敛误差的一个**理论化上界（上界）**，显式地在“同伴偏差 (Bias，即梯度方向不匹配)”与“随机方差 (Variance，即梯度噪声大)”之间进行最优权衡。该方法可自动为有害的同伴分配零权重。

*   **关键技术细节**:
    1.  **问题建模**: 目标客户端 0 执行加权 SGD 更新： $w_{t+1} = w_t - \eta g_t$，其中 $g_t = \sum_{i} \alpha^{(i)}_t g^{(i)}_t$，$\alpha_t$ 是待学习的聚合权重。
    2.  **损失函数定义**: 定义了促进目标客户端收敛的成本函数 $L_t(\alpha)$，该函数包含了两个矛盾的部分：
        *   **偏差项**: $\parallel \sum_i \alpha^{(i)} b^{(i)}_t \parallel^2$，其中 $b^{(i)}_t$ 是全局梯度与同伴梯度之差。偏差越小越好。
        *   **方差项**: $\eta L \parallel \sum_i \alpha^{(i)} g^{(i)}_t \parallel^2$，其中包含梯度噪声。方差越小越好。
        *   理论证明，最小化累积成本 $\sum E[L_t(\alpha)]$ 等价于优化目标客户端收敛速度的上界（定理 3.1）。
    3.  **自适应权重学习**:
        *   **偏差估计**: 由于真实偏差 $b^{(i)}_t$ 不可知，算法通过**指数移动平均 (EMA)** 在线估计 $\hat{b}^{(i)}_t$。
        *   **在线凸优化**: 将权重选择问题建模为一个**在线凸优化 (OCO)** 问题，使用 **Follow-The-Approximate-Leader (FTAL)** 算法来最小化由估计偏差构成的累积代理损失。FTAL 能给出对数级别的遗憾界，确保算法性能接近最优的静态权重。
    4.  **权重稀疏性与自动筛选**:
        *   理论推导的最优权重形式（引理 3.1）为：$\alpha^{(i)*} \propto [\lambda - C_1 \parallel b^{(i)} \parallel^2]_{+} / (2 C_2 (\sigma^{(i)})^2)$。这揭示了：
            *   权重与同伴的噪声方差 $(\sigma^{(i)})^2$ 成反比。
            *   当同伴偏差 $\parallel b^{(i)} \parallel^2$ 超过一定阈值时，权重被截断为零，自动排除有害同伴。
