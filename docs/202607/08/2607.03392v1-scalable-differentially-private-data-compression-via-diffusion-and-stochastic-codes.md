## 摘要
个人数据的不断收集给开发保护个人身份敏感方面的技术带来了越来越大的压力。差分隐私（DP）提供了一个具有强大形式化保证的原则性框架，并已取得实际成功。然而，发布高维数据（如图像）仍然难以实现：发布未压缩的私有化数据需要大量存储空间。同时，目前尚无有效的能压缩高分辨率数据并具有隐私保证的数据压缩方案。我们通过DP-DiPP解决了这一挑战，这是一个结合随机码与扩散模型的压缩管道。DP-DiPP非常灵活：从业者可以直接控制压缩率-隐私-效用的权衡。作为理论基础，我们将泊松私有表示（PPR）扩展用于编码隐私机制的输出。然后将其与基于扩散的有损数据压缩方法DiffC结合，得到一个差分隐私图像压缩器。我们在CIFAR-10上对私有化图像分类的实验表明，DP-DiPP显著优于基线，在保持可比较的隐私保证和效用的同时，实现了10-30倍的更好的压缩。

## Abstract
The ever-increasing collection of personal data has created mounting pressure to develop technologies that protect sensitive aspects of individual identity. Differential privacy (DP) provides a principled framework with strong formal guarantees and has already achieved practical success. However, releasing high-dimensional data, such as images, has remained elusive: releasing uncompressed privatized data requires significant storage. At the same time, no effective data compression scheme exists that can compress high-resolution data with privacy guarantees.   We address this challenge with DP-DiPP, a compression pipeline that combines stochastic codes with diffusion models. DP-DiPP is highly flexible: the practitioner has direct control over the compression rate-privacy-utility tradeoff. As the theoretical backbone, we extend the Poisson private representation (PPR) to encode the outputs of privacy mechanisms. We then combine it with DiffC, a diffusion-based lossy data compression method, to obtain a differentially private image compressor. Our experiments on privatized image classification on CIFAR-10 demonstrate that DP-DiPP significantly outperforms the baseline, achieving a 10-30 times better compression while retaining comparable privacy guarantees and utility.

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：随着个人数据收集的快速增长，保护个体隐私变得至关重要。差分隐私（DP）提供了严格的形式化保证，并已在实践中取得成功（如用户数据收集、人口普查数据发布、生成式模型防止记忆等）。然而，将DP扩展到高维数据（如图像、音频、视频）仍面临巨大挑战：
  - 直接发布私有化数据需要巨大的存储和通信成本。
  - 高维空间中为提供有意义的隐私保证而添加的噪声会严重降低数据的效用。
  - 现有工作缺乏系统研究高分辨率私有化数据的压缩问题，即如何同时保护隐私、降低存储成本并保留下游效用。
- **核心问题**：如何设计一种可扩展的差分隐私数据压缩方案，使得压缩率、隐私和效用三者之间可灵活权衡，并能在高维数据上实际运行。
- **整体含义**：本文提出DP-DiPP（Differentially Private DiffC with Poisson Processes），一种将随机码（stochastic codes）与扩散模型结合的压缩管道，首次实现了对高维图像进行可扩展的差分隐私压缩，在保持相近隐私和效用的同时将比特率降低10-30倍。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：利用随机码对差分隐私机制的输出进行联合压缩，而非先私有化再压缩。具体地：
  - 扩展泊松私有表示（Poisson Private Representation, PPR）为**步长限制的PPR**（Step-limited PPR），使其能够近似编码任意纯ε-LDP机制，并保持隐私保证、有限运行时间和接近最优的码长。
  - 将扩散模型DiffC（基于高斯扩散的有损压缩框架）中的高斯去噪目标替换为**矩匹配的拉普拉斯分布**，从而获得纯ε-LDP保证。
  - 将步长限制PPR应用于修改后的DiffC的每一步去噪过程，实现联合私有化和压缩。

- **关键技术细节**：
  - **步长限制PPR（Definition 2）**：
    - 参数：n（候选样本预算），α > 1（隐私-码长权衡参数）。
    - 编码器：在n个候选样本中，选择最小化 `V_k * (T_k / r(M_k|x))^α` 的索引K，其中 `{(M_k, T_k)}` 是泊松过程（公共随机性），`V_k` 是独立指数随机变量（仅编码器可知）。然后通过ζ分布算术码编码K。
    - 解码器：根据K从公共随机性中取出对应样本 `M_K`。
    - 隐私保证：`enc_α^n(x, Z, V)` 满足 `2αε`-LDP（定理3.1）。
    - 码长上界：`E[|enc|] ≤ ℓ_α(x) + log(ℓ_α(x) + 1) + 2`，其中 `ℓ_α = D_KL[P_{M|X}||Q_M] + log 3.56 / min{(α-1)/2, 1}`。
    - 近似质量：选择 `n ≥ 2^{ℓ_α(X)/β}` 可保证总变差距离 ≤ β（定理3.1）。
  - **矩匹配拉普拉斯机制**：将每个去噪步的高斯条件后验 `N(μ_q, σ^2 I)` 替换为拉普拉斯分布 `L(μ_q, σ/√2 * I)`，使得每一步成为纯ε-LDP机制，且密度比有界（便于PPR执行）。
  - **隐私预算分配**：通过资源约束最短路径问题选择去噪步调度，最小化总码长满足总隐私预算 `ε_total = 2α Σ ε_{t_i→t_{i-1}}`。
  - **完全因子化的CUDA实现**：利用并行计算在GPU上同时生成n个候选拉普拉斯样本并计算密度比。

- **算法流程**（文字说明）：
  1. 输入敏感图像x0，隐私参数ε，预算n，PPR参数α。
  2. 使用扩散模型和最短路径算法确定去噪步调度（从t=1到选定的最终步T_final）。
  3. 对于每一步 s→t：
     - 计算矩匹配的拉普拉斯目标分布 `L(μ_q, b)` 和提议分布 `L(μ_p, b)`。
     - 利用步长限制PPR：在公共随机性中生成n个候选样本，计算每个样本的密度比，选择最优索引K，输出压缩码字。
  4. 解码器根据共享随机性和压缩码字恢复样本，最后从T_final运行DDPM去噪到t=0得到重建图像。

## 3. 实验设计

- **数据集**：CIFAR-10（32×32×3图像）。使用预训练的DDPM模型（在CIFAR-10训练集上训练）。为避免数据泄露，将CIFAR-10测试集分成5000张训练/5000张评估图像用于分类器训练。
- **下游任务**：在私有化压缩图像上训练ResNet-56分类器（私有化后的噪声域中训练），使用分类准确率作为效用函数。

  - **基线（Privatize-then-compress）**：对图像添加独立拉普拉斯噪声（每像素纯ε-LDP），然后使用PNG无损压缩。
  - **原始DiffC（Gaussian DiffC）**：使用高斯机制且PPR参数α=∞（无隐私保证，作为消融对照）。

## 6. 主要结论与发现

- DP-DiPP显著优于私有化-再压缩基线：在相同隐私级别和分类准确率下，比特率降低10-30倍（即压缩率提升10-30倍）。
- 使用矩匹配拉普拉斯分布替换高斯分布，并采用α=2的PPR，在效用几乎不变的前提下，比特率约增加2倍，但换来了纯ε-LDP保证。
- 步长限制PPR能够在有限预算下接近最优码长，并保持隐私证书，具有理论保证和实用效率。
- 扩散模型可以作为隐私机制，通过选择合适的噪声分布（拉普拉斯）实现纯DP。

## 7. 优点

- **理论创新**：提出了步长限制PPR，解决了原始PPR运行时间可能无限的问题，并给出总变差距离、码长和隐私的严格界限。
- **方法实用性**：将随机码与扩散模型结合，利用扩散模型的渐进去噪特性分解高维压缩问题，使大规模图像压缩可行。
