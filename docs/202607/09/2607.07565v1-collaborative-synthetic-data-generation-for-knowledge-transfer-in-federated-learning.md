## 摘要
单轮联邦学习通过将训练限制在一轮内来解决联邦学习的通信开销，但在不牺牲模型质量的情况下做到这一点并非易事，尤其是在客户端数据分布存在差异时。最近的工作通过构建可迁移的合成数据集或蒸馏物，在服务器端聚合客户端知识来应对这一挑战。然而，这些方法大多缺乏正式的隐私保证，在同时实现低通信、对异构性的鲁棒性和严格隐私方面存在空白。我们提出了FedKT-CSD（基于协作式合成数据的联邦知识迁移），这是一个受神经图像压缩启发的框架，通过利用公开预训练的自动编码器作为共享潜在空间来填补这一空白。每个客户端通过单次前向传播对其私有数据进行编码，计算类别条件的潜在统计数据，并将其传输到服务器。服务器通过安全聚合聚合这些统计数据，添加校准的差分隐私噪声，并解码出一个合成数据集，用于训练全局模型和进一步的下游任务。这一设计通过构造提供了形式化的(ε,δ)-差分隐私，同时保持了客户端计算和通信的轻量级。尽管在隐私约束下运行，FedKT-CSD在不同数据集和异构性设置下与无隐私基线的性能相当甚至更优，并且可扩展到大量客户端。我们的代码可在 https://github.com/an7123/FedKT-CSD 获取。

## Abstract
One-shot federated learning (OSFL) addresses the communication overhead of federated learning by limiting training to a single round, but doing so without sacrificing model quality is non-trivial, particularly when client data distributions diverge. Recent work has addressed this challenge by aggregating client knowledge on the server through the construction of transferable synthetic datasets or distillates. However, most of these methods lack formal privacy guarantees, leaving a gap in jointly achieving low communication, robustness to heterogeneity, and rigorous privacy. We propose FedKT-CSD (Federated Knowledge Transfer via Collaborative Synthetic Data), a framework inspired by neural image compression that closes this gap by leveraging publicly pretrained autoencoders as a shared latent space. Each client encodes its private data in a single forward pass, computes class-conditional latent statistics, and transmits these to the server. The server aggregates these statistics via secure aggregation, adds calibrated differential privacy noise, and decodes a synthetic dataset for training a global model and further downstream tasks. This design provides formal $(\varepsilon,δ)$-differential privacy by construction, while keeping client-side computation and communication lightweight. Despite operating under privacy constraints, FedKT-CSD is competitive with and even outperforms non-private baselines across diverse datasets and heterogeneity settings, and scales to a large number of clients. Our code is available at: https://github.com/an7123/FedKT-CSD

## 1. 核心问题与整体含义（研究动机与背景）
联邦学习（FL）面临数据异构性导致的收敛慢和性能不佳问题。单轮联邦学习（OSFL）通过限制通信轮次来解决通信开销，但现有多数 OSFL 方法（如 FedD3、DENSE、CoBoosting、FedCVAE 等）缺乏形式化隐私保证，且客户端计算负担重。本文旨在同时实现：低通信、对数据异构的鲁棒性、以及严格的差分隐私（DP）保证。作者利用公开预训练的自编码器作为共享潜在空间，提出 FedKT-CSD 框架：客户端在单次前向传播中编码私有数据，计算类条件潜在统计量，通过安全聚合传输到服务器，服务器添加校准的 DP 噪声后解码出合成数据集用于下游任务。

## 2. 方法论
- **核心思想**：利用预训练自编码器（如 DC-AE）的编码器 \(E\) 将图像映射到低维潜在向量 \(z\)，计算类条件统计量（均值之和、外积之和、样本计数），这些统计量是加性的，可安全聚合；服务器恢复类条件高斯分布，添加 DP 噪声后解码生成合成图像数据集。
- **关键技术细节**：
  - **编码与裁剪**：客户端对每张图像 \(x\) 计算 \(z = E(x)\)，并裁剪到 \(\ell_2\) 范数 \(\le R\)（\(R = \alpha\sqrt{d}\)，数据无关）。
  - **类条件统计量**：对每个类别 \(c\)，客户端计算：均值之和 \(m_i^c\)、外积之和 \(M_i^c\)、样本数 \(n_i^c\)。
  - **安全聚合**：客户端将统计量通过安全聚合上传，服务器得到全局和：\(S_{\mu,c}, S_{\Sigma,c}, n_c\)。
  - **DP 噪声注入**：服务器对 \(S_{\mu,c}\) 和 \(S_{\Sigma,c}\) 分别添加高斯噪声，校准采用分析高斯机制（Balle & Wang, 2018），满足 \((\varepsilon,\delta)\)-DP。
  - **后处理**：服务器根据噪声和计算类条件均值 \(\mu_c^{\text{DP}}\) 和协方差 \(\Sigma_c^{\text{DP}}\)，进行偏差校正和 PSD 投影，然后从 \(\mathcal{N}(\mu_c^{\text{DP}}, \Sigma_c^{\text{DP}})\) 采样潜在向量，通过解码器 \(D\) 生成合成图像。
- **算法流程**（文字）：
  1. 每个客户端：对本地数据逐张编码（单次前向），裁剪，按类统计。
  2. 上传三要素（各客户端并行）。
  3. 服务器：安全聚合，加 DP 噪声，恢复每个类的高斯参数，采样并解码得到合成数据集 \(X^*\)。
  4. 在 \(X^*\) 上训练下游模型（如 ResNet-18）。

## 6. 主要结论与发现
- FedKT-CSD 在 (ε=10, δ=10⁻⁵)-DP 下，在所有 4 个数据集和所有异构设置上，准确率优于所有无 DP 的 OSFL 基线（提升 5–30 个百分点）。
- 对异构性和客户端数量完全不变：因为加和统计量与数据划分无关；随着客户端从 20 增加到 1000，性能几乎不变，而基线大幅下降。
- 在 DP 合成数据方法中，FedKT-CSD 在联邦设置下优于 DP-MERF、DP-NTK、DP-Kernel、DP-LoRA LDM（后者在联邦下性能崩溃）。
- 合成数据可用于下游任务：作为 pFL 的预训练数据，在 CIFAR-100 病态下比多轮 pFL 方法高 9 个点。
- 成员推断攻击 AUC = 0.501（接近随机），说明 DP 有效。

## 7. 优点
- **形式化隐私保证**：首次在 OSFL 中提供严格 (ε,δ)-DP，且隐私会计简单（单次发布）。
- **通信极低**：单轮，单客户端上传仅 KB 级别（327.5 KB），远低于模型参数（~45 MB）。
- **客户端零训练**：仅一次前向传播，无梯度计算，适合边缘设备。
- **异构不变性**：加性统计量使性能不随数据划分变化。
- **合成数据集模型无关**：生成的图像可被任何架构训练，支持多种下游任务（pFL、知识蒸馏等），无需依赖生成模型。
- **可扩展性**：支持 1000+ 客户端，性能无降解。

## 8. 不足与局限
- **依赖类标签**：假设标签已知且公开，无法处理无监督或自监督场景；未保护类别归属（虽然可扩展保护）。
- **单一高斯假设**：每个类仅用一个高斯分布建模，对于多模态或复杂类内分布能力有限；混合高斯模型（GMM）虽更灵活但难以加性聚合。
- **小样本类性能差**：当类样本数 \(n_c\) 很小，DP 噪声过大，均值有效噪声 \(\sigma_\mu / n_c\) 显著，影响生成质量。
- **实验缺少真实世界异构（如特征分布偏移）**：仅考察了标签分布异构（Dirichlet 和病态），未覆盖特征非独立同分布（如不同设备相机差异）。
- **隐私预算设置单一**：主要使用 ε=10，未充分探索低预算（ε<1）下的性能（仅消融中展示了 ε=1 时约 25% 准确率）。
- **无客户端模型隐私保护细节**：安全聚合假设服务器诚实，未讨论恶意客户端或服务器勾结威胁。
- **合成图像质量一般**：对于较复杂数据集（CIFAR-100、ImageNette）生成的图像视觉质量不高，但分类性能尚可。
