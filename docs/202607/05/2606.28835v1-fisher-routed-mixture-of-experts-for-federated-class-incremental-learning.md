## 摘要
联邦学习（FL）作为一种有前景的分布式学习范式涌现。然而，将FL扩展到类增量学习场景面临独特挑战：1）共享模型过载带来的容量冲突与灾难性遗忘，2）非独立同分布（Non-IID）数据导致的异质性，以及3）同步类别错位。本文提出基于Fisher路由的混合专家模型用于联邦类增量学习（FedFMX），这是一个通过跨客户端的自适应专家专业化来应对这些挑战的新框架。关键见解是将每个样本路由到一个共同优化知识获取与保持的专家子集。具体而言，我们引入Fisher路由专家评分（FRES）模块，通过基于Fisher的稳定性代价和基于梯度的可塑性增益来估计专家重要性。然后，通过量化边际贡献设计自适应专家选择（AES）模块，以实现自适应专家子集确定。最后，通过路由感知正则化（RAR），实现负载均衡与高效FL训练。我们从理论上证明了$\mathcal{O}(T^{-1})$的收敛速率。在多个基准上与最先进方法相比的大量实验表明，FedFMX具有优越性。

## Abstract
Federated Learning (FL) emerged as a promising distributed machine learning paradigm. However, extending FL to the class incremental learning scenarios introduces unique challenges: 1) Capacity conflict and catastrophic forgetting from the shared model overloading, 2) Heterogeneity from Non-Independent and Identically Distributed (Non-IID) data, and 3) Synchronized class misalignment. In this paper, we propose \textbf{F}isher-Routed \textbf{M}i\textbf{X}ture of Experts for \textbf{Fed}erated Class-Incremental Learning (\textsc{FedFMX}), a novel framework to address these challenges via adaptive expert specialization across clients. The crucial insight is to route each sample to an expert subset that jointly optimizes knowledge acquisition and retention. Specifically, we introduce a Fisher-Routed Expert Scoring (FRES) module to estimate expert importance via Fisher-based stability cost and gradient-based plasticity gain. Then, we design an Adaptive Expert Selection (AES) module by quantifying marginal contributions for adaptive expert subset determination. Finally, by the routing-aware regularization (RAR), we achieve load balance and efficient FL training. We theoretically prove the $\mathcal{O}(T^{-1})$ convergence rate. Extensive experiments on multiple benchmarks compared with state-of-the-art methods demonstrate the superiority of \textsc{FedFMX}.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：联邦学习（FL）假设客户端标签分布静态且一致，但现实中客户端数据动态演化，不断出现新类别（Class-Incremental）。联邦类增量学习（FCIL）面临三个根本挑战：
  1. **容量冲突与灾难性遗忘**：共享全局模型被异构数据不断过载，旧知识被覆盖。
  2. **统计异质性**：客户端Non-IID数据导致梯度方向不一致，损害全局收敛。
  3. **同步类别错位**：不同客户端在不同阶段接收到新类，导致标签空间时间不一致，加剧模型漂移。
- **整体含义**：现有方法将全局模型视为单一整体，缺乏细粒度机制来调节稳定性（保留旧知识）与可塑性（学习新类）。本文旨在通过引入专家模块化和自适应路由来解耦复杂动态，平衡知识获取与保留，实现稳健高效的FCIL。

### 2. 方法论：核心思想、关键技术细节

- **核心思想**：将模型分解为共享骨干网络和多个专家子空间，每个样本被路由到一个动态选择的专家子集，该子集共同优化知识获取与保留。
- **关键技术细节**：
  1. **Fisher路由专家评分（FRES）模块**：
     - 对每个候选专家，在激活前计算两个指标：
       - **稳定性代价** $\Delta_c^{(k)} = \frac{1}{2} \| \text{diag}(\tilde{F}_k)^{1/2} \tilde{g}_k \|_2^2$：惩罚在历史重要参数方向上产生大梯度的专家。
       - **可塑性增益** $\Delta_g^{(k)} = \| g_k \|_2^2 / (Tr(\tilde{F}_k) + \epsilon)$：衡量专家在相对平坦参数区域安全吸收新信息的能力。
     - 使用对角Fisher矩阵（通过指数移动平均维护平方梯度）近似参数重要性，无需额外反向传播。
  2. **自适应专家选择（AES）模块**：
     - 将专家选择建模为**合作博弈**，定义价值函数 $v(S) = -\ell_S(x,y) - \alpha \sum_{k \in S} \Delta_c^{(k)} + (1-\alpha)\Delta_g(S)$。
     - 通过计算每个专家的**夏普利值**（Shapley value）量化其边际贡献，然后按贡献降序排列专家，迭代添加专家直到边际增益增量小于阈值 $\mu$，从而动态确定最佳专家子集 $\mathcal{K}_b$。
  3. **路由感知正则化（RAR）**：
     - **负载均衡正则化** $\mathcal{L}_{bal}$：熵正则化项，抑制尖锐的路由分布，防止少数专家过载。
     - **蒸馏门控机制** $\mathcal{L}_r$：将夏普利值转化为软教师分布 $q(k|x_b)$，训练轻量级门控网络 $g_\eta$ 输出学生分布 $p(k|x_b)$，通过KL散度对齐，使得推理时无需计算昂贵的边际贡献。
  4. **总体损失**：$\mathcal{L} = \mathcal{L}_{task} + \beta \mathcal{L}_{bal} + \gamma \mathcal{L}_r$。在激活集 $\mathcal{K}_b$ 内，各专家权重 $w_k \propto \max(\phi_k, 0)$，抑制噪声专家。
- **算法流程**（文字说明）：每个全局轮次中，服务器分发全局模型（含共享骨干和专家池）给部分客户端；客户端本地训练：对每个batch，通过FRES计算每个专家的稳定性代价和可塑性增益；AES基于夏普利值动态选择激活专家子集；使用RAR损失（含负载均衡和蒸馏）更新模型；服务器聚合所有客户端的模型更新。
