## 摘要
联邦微调（FFT）已成为一种在保护隐私的前提下协作适配大语言模型（LLMs）的范式。基于联邦学习，FFT 允许分布式代理通过聚合本地 LLM 更新来共同优化共享的预训练 LLM，而无需共享本地原始数据。然而，基于 FFT 的 LLM 仍然容易受到模型操控威胁，即对抗性参与者上传经过操控的 LLM 更新，从而破坏聚合过程并降低全局 LLM 的性能。在本文中，我们提出了一种针对基于 FFT 的 LLM 的增强型模型操控（AugMP）策略。具体而言，我们设计了一个新颖的图表示学习框架，用于捕获良性 LLM 更新之间的特征相关性，以指导恶意更新的生成。为了提高操控的有效性和隐蔽性，我们开发了一种基于增广拉格朗日对偶公式的迭代操控算法。通过该公式，恶意更新被优化以嵌入对抗性目标，同时保留类似良性的参数特征。在多个 LLM 主干网络上的实验结果表明，AugMP 策略在所有竞争基准中实现了最强的操控性能，使全局 LLM 准确率降低了高达 26%，并使本地 LLM 代理的平均准确率降低了高达 22%。同时，AugMP 与良性更新保持了高度的统计和几何一致性，使其能够规避传统的基于距离和相似性的防御方法。

## Abstract
Federated fine-tuning (FFT) has emerged as a privacy-preserving paradigm for collaboratively adapting large language models (LLMs). Built upon federated learning, FFT enables distributed agents to jointly refine a shared pretrained LLM by aggregating local LLM updates without sharing local raw data. However, FFT-based LLMs remain vulnerable to model manipulation threats, in which adversarial participants upload manipulated LLM updates that corrupt the aggregation process and degrade the performance of the global LLM. In this paper, we propose an Augmented Model maniPulation (AugMP) strategy against FFT-based LLMs. Specifically, we design a novel graph representation learning framework that captures feature correlations among benign LLM updates to guide the generation of malicious updates. To enhance manipulation effectiveness and stealthiness, we develop an iterative manipulation algorithm based on an augmented Lagrangian dual formulation. Through this formulation, malicious updates are optimized to embed adversarial objectives while preserving benign-like parameter characteristics. Experimental results across multiple LLM backbones demonstrate that the AugMP strategy achieves the strongest manipulation performance among all competing baselines, reducing the global LLM accuracy by up to 26% and degrading the average accuracy of local LLM agents by up to 22%. Meanwhile, AugMP maintains high statistical and geometric consistency with benign updates, enabling it to evade conventional distance- and similarity-based defense methods.

随着大语言模型（LLM）在边缘计算和分布式网络中的部署，**联邦微调（Federated Fine-Tuning, FFT）**成为一种主流的隐私保护适配方案。它允许各参与方仅通过交换低秩自适应（LoRA）参数更新来协作训练模型。
然而，该研究指出 FFT 存在严重的**模型操控（Model Manipulation）**风险。攻击者可以作为合法客户端加入，上传精心构造的恶意更新。现有的防御机制主要依赖欧几里得距离或余弦相似度来检测异常，而本论文的研究动机在于证明：攻击者可以通过学习良性更新的特征分布，生成既能大幅破坏全局模型性能，又能完美规避现有几何检测手段的“隐形”恶意更新。

### 2. 方法论：核心思想与关键技术
论文提出了名为 **AugMP（Augmented Model Manipulation）** 的攻击策略，其核心流程如下：
*   **图表示学习（GRL）框架**：攻击者观察部分良性客户端的更新，将其构建为特征相关图。利用**变分图自编码器（VGAE）**学习这些更新之间的内在结构和特征相关性。
*   **图频谱变换（GST）**：通过对学习到的图进行拉普拉斯分解和图傅里叶变换，在频谱域重构特征矩阵，从而生成初始的恶意更新，确保其在特征空间上与良性更新高度一致。
*   **增广拉格朗日对偶优化**：将攻击建模为一个受限优化问题。目标是最大化全局训练损失（破坏性能），同时受到几何约束（距离和相似度必须在防御阈值内）。通过迭代更新对偶变量和惩罚参数，精细化调整恶意更新的方向。

### 3. 实验设计
*   **数据集**：使用了两个主流文本分类基准：**AG News**（4类）和 **Yahoo! Answers**（10类）。
*   **模型主干（Backbones）**：涵盖了不同架构和规模的模型，包括 **DistilBERT**（67M，Encoder-only）、**Pythia**（160M，Decoder-only）和 **Qwen2.5**（500M，Decoder-only）。
*   **对比方法（Benchmarks）**：
    *   **ALIE**：通过估计良性更新的标准差来偏移均值的攻击。
    *   **RMP（Gaussian Random Model Poisoning）**：基于高斯分布采样的随机模型投毒。
    *   **Benign Setting**：无攻击的正常联邦微调作为基准。

*   **硬件环境**：实验在一台运行 Linux 的工作站上完成，配备了 **1 张 NVIDIA A100 GPU (80GB 显存)**。

*   **客观性**：实验不仅关注全局模型的性能下降，还分析了攻击对本地代理的影响，并详细展示了恶意更新在距离和相似度分布上的统计特性，证明了其规避防御的真实能力。整体实验设计较为全面且客观。

### 6. 主要结论与发现
*   **破坏力极强**：AugMP 在 Pythia 模型上使全局准确率下降了 **26%**，在其他模型上也表现出显著的性能削弱。
*   **高隐蔽性**：与 ALIE 和 RMP 明显偏离良性分布不同，AugMP 生成的更新在欧氏距离和余弦相似度上与良性更新几乎重合，现有的基于统计异常的防御完全失效。
*   **传播效应**：由于联邦学习的广播机制，受污染的全局模型会回传给所有良性客户端，导致本地代理的平均准确率也下降了高达 **22%**。
*   **可解释性破坏**：通过案例研究发现，受攻击的模型虽然仍能生成流利的解释，但其逻辑会发生扭曲（例如将商业新闻强行解释为科技新闻）。

### 7. 优点
*   **攻击视角新颖**：首次将图表示学习引入 FFT 的模型操控领域，捕捉了高维参数间的复杂相关性。
*   **数学严谨性**：利用增广拉格朗日法解决了非凸约束优化问题，使攻击在理论上更具鲁棒性。
*   **开源贡献**：提供了完整的 PyTorch 实现代码，有助于安全社区进行后续的防御研究。

### 8. 不足与局限
*   **模型规模限制**：实验最高仅测试了 500M 参数的模型（Qwen2.5-0.5B），对于 7B 或 70B 等更大规模的 LLM，图构建的计算开销和攻击效果尚待验证。
*   **防御假设较窄**：主要针对基于几何距离的防御，未探讨针对基于鲁棒聚合（如 Median, Trimmed Mean）或基于验证集检测的防御效果。
*   **攻击者知识假设**：假设攻击者能观察到部分良性更新，在某些高度加密（如使用同态加密或安全多方计算）的联邦学习场景下，该攻击的前提条件可能难以满足。
