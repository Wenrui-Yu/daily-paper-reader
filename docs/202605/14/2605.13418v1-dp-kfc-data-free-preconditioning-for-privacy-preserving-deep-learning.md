## 摘要
差分隐私优化面临一个根本性的几何失配问题：深度网络具有高度各向异性的损失景观，而 DP-SGD 注入的是各向同性噪声。二阶预条件化可以解决这一问题，但估计曲率通常需要私有数据（消耗隐私预算）或公共数据（引入分布偏移）。我们证明了 Fisher 信息矩阵可以解耦为架构敏感性（可通过合成噪声恢复）和输入相关性（可从特定模态的频率统计中近似）。我们提出了 DP-KFC，它通过使用结构化合成噪声探测网络来构建 KFAC 预条件算子，既不需要私有数据也不需要公共数据。实验表明，在强隐私机制（$\varepsilon \leq 3$）下，DP-KFC 在多种模态上始终优于 DP-SGD 和自适应基准方法。DP-KFC 的性能与基于私有数据的预条件算子相当，而基于公共数据的变体性能下降高达 4.8%，这表明可以在不消耗隐私预算或引入分布偏移的情况下估计曲率。这使得在受监管约束导致数据稀缺的专业领域（如医疗应用）中进行隐私保护学习成为可能。

## Abstract
Differentially private optimization suffers from a fundamental geometric mismatch: deep networks have highly anisotropic loss landscapes, yet DP-SGD injects isotropic noise. Second-order preconditioning can resolve this, but estimating curvature typically requires private data (consuming privacy budget) or public data (introducing distribution shift). We show that the Fisher Information Matrix decouples into architectural sensitivity, recoverable via synthetic noise, and input correlations, approximable from modality-specific frequency statistics. We propose DP-KFC, which constructs KFAC preconditioners by probing networks with structured synthetic noise, requiring neither private nor public data. Empirically, DP-KFC consistently outperforms DP-SGD and adaptive baselines across diverse modalities in strong privacy regimes ($\varepsilon \leq 3$). DP-KFC matches private-data preconditioners while public-data variants degrade by up to $4.8\%$, showing that curvature can be estimated without consuming privacy budget or introducing distribution shift. This enables privacy-preserving learning in specialized domains (e.g., medical applications) where regulatory constraints make data scarce.

*   **现有方案的缺陷**：
    *   **私有数据估计**：使用私有数据估计曲率矩阵会消耗宝贵的隐私预算（$\varepsilon$），且噪声化的预条件算子往往不稳定。

### 2. 方法论：DP-KFC
*   **核心思想**：论文发现 Fisher 信息矩阵（FIM）可以解耦为**架构敏感性**（由网络权重和结构决定）和**输入相关性**（由模态统计特性决定）。通过合成噪声探测网络，可以提取出这些内在的几何特征。
*   **关键技术细节**：
    *   **合成结构化探测（Synthetic Probing）**：不使用真实数据，而是根据模态特性生成合成输入。
        *   **图像领域**：生成符合 $1/f^\alpha$ 功率谱的**粉红噪声（Pink Noise）**，模拟自然图像的频率分布。
        *   **文本领域**：生成符合齐夫定律（Zipfian）频率分布的随机 Token 序列，并保留特殊的结构化 Token（如 [CLS], [SEP]）。
    *   **KFAC 预条件化**：利用 Kronecker 因子分解近似 Fisher 矩阵。通过合成数据进行前向和后向传播，估计每一层的激活协方差因子 $A$ 和梯度协方差因子 $G$。
    *   **先缩放后私有化（Scale-then-Privatize）**：在应用隐私机制（裁剪和加噪）之前，先用合成数据生成的预条件算子对梯度进行坐标变换，使梯度空间趋于各向同性。
*   **算法流程**：
    1.  生成合成批次数据。
    2.  通过当前模型进行前向/后向传播，计算层级 KFAC 因子。
    3.  对因子进行特征分解，计算正则化的逆平方根矩阵。
    4.  在私有训练步中，先用该矩阵变换私有梯度，再进行标准 DP-SGD 的裁剪和加噪。

### 6. 主要结论与发现
*   **性能优越性**：在强隐私机制（$\varepsilon \leq 3$）下，DP-KFC 始终优于 DP-SGD。在 MNIST 上比 DP-SGD 提升约 2.5%，在 IMDB 上提升约 2.7%。
*   **架构主导论**：实验证明曲率矩阵的特征值衰减主要由网络架构决定，合成噪声能有效捕捉这种衰减，其效果在视觉任务中几乎等同于私有数据。
*   **鲁棒性**：在领域偏移（如医疗影像）场景下，公共数据预处理会因“负迁移”导致性能下降，而 DP-KFC 表现稳定，甚至优于基于错误公共数据的方案。

### 7. 优点
*   **隐私零成本**：预条件算子的构建完全不消耗隐私预算。
*   **无数据依赖**：解决了专业领域缺乏公共代理数据的痛点。
*   **超参鲁棒性**：DP-KFC 极大地放宽了裁剪阈值（Clip Norm）和学习率的耦合关系，使模型更容易调优。
*   **理论支撑强**：结合了均值场理论（Mean-Field Theory）和随机矩阵理论，对为何合成噪声有效给出了深刻解释。

### 8. 不足与局限
*   **NLP 表现稍逊**：在文本任务中，合成 Token 探测虽然优于 DP-SGD，但仍落后于公共数据预处理。这是因为文本数据分布在极低维的流形上，简单的频率随机采样难以完全模拟真实文本的激活特性。
*   **计算开销**：虽然在 DP 语境下合理，但 2.2 倍的减速在超大规模模型训练中仍是不小的负担。
*   **深度层对齐失效**：在网络的极深层，特征向量的方向高度依赖于私有标签，合成噪声只能恢复特征值的尺度（Scale），无法完美恢复方向（Direction）。
